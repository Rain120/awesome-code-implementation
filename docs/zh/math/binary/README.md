## 什么是二进制?

十进制计数是使用 `10`作为基数, 二进制计数是使用 `2`作为基数, 二进制的数位就是 `2^n`的形式。

## 为什么使用二进制?

二进制的数据表达具有抗干扰能力强、可靠性高的优点; 二进制非常适合做逻辑运算。

## 位运算

移位操作

- 左移

```
value << 1 = Math.floor(value * 2)

value << n = Math.floor(value * 2 ^ n)
```

- 右移

```
value >> 1 = Math.floor(value / 2)

value >> n = Math.floor(value / 2 ^ n)
```

逻辑操作

- 与 `&`

见 `0` 出 `0`

- 或 `|`

见 `1` 出 `1`

- 非 `~`

`0 -> 1, 1 -> 0`

- 异或 `^`

同为 `0`, 异为 `1`
