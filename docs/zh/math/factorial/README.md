## 定义

一个正整数 `n` 的阶乘 (写作 `n!`), 就是所有小于等于 `n`的正整数的乘积

**数学公式**

$$ f(0) = 1 $$

$$ f(n) = n * (n - 1) * ... * 1 $$

Eg:
```text
n! = 1 * 2 * ... * (n - 1)
```
